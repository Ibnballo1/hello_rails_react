To get a local copy up and running, follow these steps.
### Prerequisites

In order to run this project you need:


```sh
    rails db:create
    rails server
    ./bin/dev
```

### Setup

Clone the repository

```
  git clone git@github.com:Ibnballo1/hello_rails_react.git
```
Navigate to the folder
